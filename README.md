# What-is-Go
این ریپوزیتوری شامل یک مقاله جامع در مورد زبان برنامه‌نویسی Golang است که به تاریخچه، ویژگی‌ها، کاربردها، و نمونه‌های کد این زبان پرداخته می‌شود. این مقاله به زبان فارسی نوشته شده و به‌طور کامل به تحلیل و بررسی امکانات و ویژگی‌های Golang، مزایای آن نسبت به سایر زبان‌ها و کاربردهای واقعی این زبان می‌پردازد.
